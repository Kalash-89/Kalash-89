# Kalash Singh — Portfolio

A single-page personal portfolio site for **Kalash Singh** — Computer Science Engineering student, aspiring Software Developer & Data Analyst.

Built as a static HTML/CSS/JS page (no build step, no dependencies) — open `index.html` in any browser or deploy it as-is.

---

## ✨ Overview

The site presents Kalash's background, projects, and skills in a bold, high-contrast design:

- **Hero** — name, role, short intro, a stat strip (year, projects built, graduation year), and a monogram graphic in place of a photo.
- **Selected Projects** — three featured data-analytics projects (IPL Analysis Dashboard, Fundraising Outreach Analysis, Deloitte Manufacturing Analysis).
- **Education & Skills** — degree details plus grouped skill pills (languages, web dev, data analytics, tools).
- **How I Work** — a five-step process (Understand → Plan → Build → Validate → Deliver).
- **Quote** — a short personal statement styled as a pull-quote.
- **Contact** — links to LinkedIn, GitHub, LeetCode, CodeChef, and HackerRank, plus a code-snippet-styled summary card.

## 🎨 Design

| | |
|---|---|
| **Style** | Bold, high-contrast, red-on-black editorial design |
| **Fonts** | [Anton](https://fonts.google.com/specimen/Anton) (display), [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) (headings), [Caveat](https://fonts.google.com/specimen/Caveat) (script accents), [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) (labels/code), [Inter](https://fonts.google.com/specimen/Inter) (body) |
| **Colors** | Near-black background, off-white ink, signature red accent (`#e2261a`) |
| **Layout** | CSS Grid, fully responsive down to mobile |
| **Accessibility** | Respects `prefers-reduced-motion`, visible focus states, semantic HTML |

No external CSS/JS frameworks — everything is hand-written in a single `index.html` file for simplicity and fast load times.
